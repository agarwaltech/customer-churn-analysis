# Customer Churn Analysis & Prediction

## Overview
This project focuses on analyzing customer data to understand churn behavior and build a machine learning model to predict churn probability.

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- SQL
- Power BI

## Workflow
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature selection
- Churn prediction using machine learning
- Visualization using Power BI
